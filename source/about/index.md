## 这个博客是什么

<strong>喵喵鱼塘</strong>是 <strong>Foam</strong> 的个人技术博客,记录一名<strong>全栈开发者(以后端为主)</strong>在真实工程中的技术积累与思考。

内容定位是「**实战笔记 + 系统化教程**」两者兼有:既有开发中随手记录的问题解决、避坑总结、工具类代码,也有对某一主题的深度解析与体系化长文。目标读者是**后端 / 全栈工程师**,以及想系统了解 Java 生态和 AI 应用落地的开发者。

## 主要写什么

博客当前共 88 篇文章,按主题分为以下方向(括号内为文章数量):

- **工程实践(19 篇)**:最佳实践、代码简化与质量、设计模式、AOP、自定义注解、云原生运维等日常开发经验
- **技术随笔(17 篇)**:开发中的问题记录、工具技巧、面向对象与基础知识梳理
- **AI 与大模型(15 篇)**:大语言模型(LLM)、RAG 检索增强生成、向量数据库、Milvus、Embedding、LangChain4j、Spring AI、MCP、AIGC 等 AI 应用开发全流程
- **Java 新特性(11 篇)**:从 Java 8 到 Java 24 的版本演进,Lambda、Stream、模式匹配、虚拟线程、结构化并发、字符串模板等特性深度解析
- **数据库(10 篇)**:MySQL 性能优化、索引与查询优化、分库分表、ShardingJDBC、Redis、分布式锁
- **性能优化(6 篇)**:JVM 调优、GC、高并发、P90/P95/P99 性能指标、Netty 高性能网络编程
- **中间件与架构(5 篇)**:Netty、状态机、事件驱动、秒杀架构、MinIO 对象存储
- **AI 编程工具(4 篇)**:Cursor、GitHub Copilot、AI 代码审查等 AI 辅助开发实践
- **前端开发(1 篇)**:现代化前端相关
